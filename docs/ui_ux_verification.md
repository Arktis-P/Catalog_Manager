## 키보드와 접근성

- [x] 모든 상단 메뉴에 공통 `focus-visible` 규칙이 적용된다.
- [x] 작업 패널 접기 버튼이 native button이며 `aria-expanded`를 제공한다.
- [x] resize separator가 방향키·Home·End·Enter와 현재/최소/최대 폭 ARIA를 제공한다.
- [x] V2 방향키 이동 시 현재 카드와 실제 DOM 포커스가 함께 이동한다.
- [x] 입력 필드 전역 Review 단축키 예외가 유지된다(코드·빌드 확인).
- [x] 카드와 작업 상태가 색상 외 텍스트로도 전달된다.
- [x] `prefers-reduced-motion`에서 변형·애니메이션을 제거한다.

## 화면별 수동 확인

- [x] 기본 뷰포트: 메뉴 그룹과 카드 그리드가 정상 배치된다.
- [x] 1000px: 메뉴가 세로 배치로 전환되고 가로 오버플로가 없다.
- [x] 작업 없음: 52px 접힌 요약 모드와 수동 펼침이 동작한다.
- [ ] 작업 있음: 실제 실행·대기·실패·완료 작업으로 수치 일치 확인 필요.
- [x] Review V2: 진행률, 남은 수, 필터, clean/dirty 상태가 보인다.
- [x] Review 모드 탭: 세 모드의 한국어 명칭·설명과 기존 URL을 확인했다.

## 사용자 확인 필요

- 기본 작업 패널 접힘 상태가 실제 사용 패턴에 적합한지

- V2에서 레이팅 4 비권장 안내 강도가 적절한지
